# FitPulse — Landing Page de Personal Trainer

Landing page de alta energia para personal trainer, desenvolvida por **Vinicius Felipe** como projeto de portfólio. Destaque para a calculadora de IMC funcional com recomendação personalizada.

## 🔗 Demo
> Após o deploy: `https://seu-usuario.github.io/fitpulse`

## Sobre o Projeto
Landing page completa para um personal trainer, com programas por objetivo, seção de resultados antes/depois, calculadora de IMC interativa e planos com pricing cards.

## Tecnologias
- HTML5 Semântico
- CSS3 (Custom Properties, Grid, Flexbox, Keyframe Animations)
- JavaScript Vanilla (ES6+)
- Oswald (Google Fonts) para títulos de impacto

## Funcionalidades
- Navbar com animação hambúrguer → X e menu mobile responsivo
- Hero com imagem de fundo e cards flutuantes animados (keyframes)
- Contadores animados de estatísticas (Intersection Observer)
- 4 cards de programas por objetivo (musculação, emagrecimento, condicionamento, online)
- Seção de resultados com layout antes/depois side-by-side
- **Calculadora de IMC funcional**: calcula o índice e exibe classificação + recomendação de programa personalizada por resultado
- 3 planos de preço com features incluídas/excluídas
- Seção "Sobre" com certificações e contadores
- Depoimentos de clientes
- Formulário de avaliação gratuita com máscara de telefone
- Botão flutuante do WhatsApp
- SEO completo: meta description, Open Graph, Twitter Card, canonical URL

## Deploy no GitHub Pages

### Passo a passo:
1. Crie um repositório chamado `fitpulse`
2. Na pasta do projeto:
   ```bash
   git init
   git add .
   git commit -m "feat: landing page FitPulse"
   git branch -M main
   git remote add origin https://github.com/SEU-USUARIO/fitpulse.git
   git push -u origin main
   ```
3. No GitHub: **Settings → Pages → Source: main / root → Save**

> **Dica:** Para publicar todos os projetos de uma vez, considere criar um repositório único chamado `portfolio` e colocar cada projeto em sua própria pasta — o GitHub Pages servirá cada `index.html` corretamente.
